环境搭建

安装Node.js

如果自己的电脑是64位则下载64位，是32位下载32位（一般都是64位）

安装界面一路点next就行了

之后再验证一下是否安装成功

Windows系统中快捷键win+r，输入cmd回车

在cmd中依次输入以下代码

node -v 输完回车

npm -v输完回车

如果出现版本信息则安装成功

切换镜像源

因为npm是国外的镜像源，使用npm下载速度很慢，所以我们换成国内的淘宝源

输入以下代码进行更换

npm install -g -cnpm --registry=https://registry.npm.taobao.org

安装完成之后输入以下代码

cnpm 回车
cnpm -v 回车

出现以下信息表示安装成功

安装git

首先我们需要在GitHub上注册一个账户

之后下载git

下载完打开文件进行安装，只有以下图片这里要勾选一下，其他的默认即可

安装完之后会多出这三个软件

之后配置环境变量

右键此电脑，选择属性

点击高级系统设置

选择环境变量

在环境变量中找到Path

点击新建，输入以下路径（bin目录和cmd目录之前的路径为你git的安装目录）

之后一路点确定就行了

至此，git安装完毕，在cmd中输入以下代码验证

git --version

Git配置

创建SSH

打开Git Bash，设置你的名字和email

依次输入以下代码并回车

Code

接下来创建ssh文件，输入以下代码（这里的-C是大写），之后按照提示回车就行了

ssh-keygen -t rsa -C “Youremail@xxx.com”

之后你的C盘中的C:\Users\Administrator.ssh目录下面会有两个文件

连接SSH

打开这个网址:https://github.com/settings/keys

选择New SSH Key

Title名随便写，之后到刚刚那个.ssh文件中用记事本打开id_rsa.pub文件，复制里面的内容粘贴到Key中

然后点Add SSH Key就可以了

然后在Git Bash里面输入以下代码

ssh -T git@github.com

它会叫你输入yes/no，输入yes，之后输入密码就结束了，这样就连上了github了

这样Git也配置完成了

安装Hexo

全局安装Hexo

在cmd中输入以下代码并回车

cnpm install -g hexo-cli

输入hexo -v来验证是否安装成功

创建blog文件夹

在cmd中切换到想要创建文件夹的目录，之后输入mkdir blog 来创建blog文件夹

之后我们就可以在那个盘目录找到blog文件夹了

初始化Hexo

我们到刚刚那个blog文件夹路径栏中输入cmd回车

这一步我在这里遇到了网络较慢，克隆失败等问题，经过一番折腾找到了解决办法

网络较慢可以为git设置代理解决

Code

这里大多数电脑都是1080端口，如果你的电脑不是1080端口可以自行更改

接下来输入hexo init

hexo init失败的解决办法可以把blog文件夹删除，再来一遍，知道成功为止

完成之后在cmd中输入hexo s回车

之后它给了一个地址：http://localhost:4000

进入浏览器输入这个地址就可以进入Hexo博客来

至此Hexo博客就初步搭建完成了

创建第一篇文章

在cmd中切换到blog文件夹下，或者直接在blog文件夹下打开cmd

在cmd中输入hexo n "我的第一篇文章"

之后我们在blog文件夹下面的source文件夹下面的_posts文件夹下面就可以发现我们创建的第一篇文章了，不过我们可以发现这个文件类型是md，所以我们可以使用md语法来编辑文章

创建好文章之后我们继续在cmd中输入hexo clean && hexo g来清除缓存和生成文件

之后输入hexo s来启动本地服务器得到这个地址：http://localhost:4000

在浏览器输入那个地址就可以进入博客界面，然后就可以看到刚刚新建的那个文章了

Description可以随便写，比如我的第一个博客，其他的不用点，然后直接点Create repository就创建完成了

之后回到cmd，安装git插件cnpm install --save hexo-deployer-git

这是Hexo的主题地址：https://hexo.io/themes/

在刚刚那个cmd中输入以下代码，注意路径是在blog文件夹下

git clone https://github.com/TriDiamond/hexo-theme-obsidian.git themes/obsidian

下载完之后可以在blog下面的themes文件夹下面找到obsidian，在obsidian文件夹下面就有相关的配置文件
